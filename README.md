🚀 Multi Utility Web App

A modern and responsive Multi Utility Web Application developed using HTML, CSS, and JavaScript.
This project combines multiple useful tools and games into one platform with a clean UI, Dark/Light mode, and mobile-friendly design.

✨ Features
🧮 Calculator
❓ Quiz Game
🌦 Weather App
📘 Dictionary App
🐍 Snake Game
🌙 Dark & Light Mode
📱 Fully Responsive Design
🔊 Sound Effects in Snake Game
💾 Local Storage Support (History & High Score)
🛠 Technologies Used
HTML5
CSS3
JavaScript
API Integration
Local Storage
📱 Responsive Design

The project is fully optimized for:

Mobile Phones
Tablets
Desktop Screens
🎯 Project Objective

The main objective of this project is to provide multiple daily-use utilities and entertainment features in a single web application with a simple and user-friendly interface.
